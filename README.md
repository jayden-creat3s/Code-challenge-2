# Trapathon Party Guest List

Welcome to the **Trapathon Party Guest List** project! This web application allows you to manage your guest list for the ultimate Trapathon party. Add guests, track attendance, and get ready for an unforgettable celebration!

- [Features](#features)
- [Technologies Used](#technologies-used)

## Features
- **User -Friendly Interface**: Easily add guests to your party list.
- **Dynamic Guest List**: View the total number of guests and their categories (Friend, Family, Colleague).
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.
- **Interactive Elements**: Engaging animations and visual effects to enhance user experience.
- **Toast Notifications**: Get instant feedback when guests are added.

## Technologies Used
- **HTML**: Structure of the web application.
- **CSS**: Styling and layout, utilizing modern CSS features like gradients and flexbox.
- **JavaScript**: Dynamic functionality for managing the guest list and handling user interactions.
